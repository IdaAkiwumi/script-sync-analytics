**Script-Sync Analytics** is a narrative ROI engine designed for Studio Executives, Creative Producers, and Narrative Strategists. While writers focus on the *art* of the script, Script-Sync focuses on the *viability*, transforming raw movie datasets into actionable product roadmaps.

---

### ✨ Why Script-Sync Analytics?

In a high-stakes production environment, "gut feeling" isn't enough to greenlight a $100M project. **Script-Sync** provides the analytical rigor (Saturn in Capricorn) to back up creative expansion (Sun in Gemini) by providing:

* **Sentiment ROI Mapping:** Uses NLP (Natural Language Processing) to score the "frustration" and "empathy" levels of narrative themes against historical market success.
* **Genre Friction Analysis:** Visualizes market saturation vs. audience appetite, identifying "Blue Ocean" opportunities for original IP.
* **The "Akiwumi-ized" UX:** Built by a **Lead Product Designer**, the dashboard prioritizes WCAG 2.1 accessibility and high-contrast data storytelling for rapid executive decision-making.
* **Predictive Roadmap Logic:** Automatically flags narrative archetypes—like "Legal Thriller" or "Action Comedy"—that are currently trending in global searches.

---

### 🛠️ Tech Stack & Key Features
* **Data Visualization:** Powered by `Plotly` and `Pandas` for interactive, deep-dive scatter plots and trend metrics.
* **NLP Engine:** Integrated `TextBlob` logic to quantify the emotional resonance of narrative datasets.
* **Strategic UI:** Custom Streamlit components designed for "Manager View" metrics and "Tactical" data exploration.
* **State Management:** Optimized caching for large-scale movie metadata processing.

### 🚀 Getting Started

#### **For Industry Professionals (Web Access):**
Visit the live deployment on **Streamlit Community Cloud** (Link above) to analyze current movie trends and narrative performance metrics.

#### **For Developers (Local Build):**
1.  **Clone the Repo:**
    ```bash
    git clone [https://github.com/IdaAkiwumi/script-sync-analytics.git](https://github.com/IdaAkiwumi/script-sync-analytics.git)
    ```
2.  **Install Dependencies:**
    ```bash
    pip install streamlit pandas plotly textblob
    ```
3.  **Launch the Engine:**
    ```bash
    streamlit run scriptsync_app.py
    ```
